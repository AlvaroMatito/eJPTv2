# eJPTv2

Repositorio con mis apuntes estructurados durante la preparación de la certificación **eJPTv2 (INE Security)**.

El objetivo de este repositorio es documentar conceptos, herramientas y técnicas clave utilizadas en entornos de laboratorio y formación en ciberseguridad ofensiva.

---

## 🎯 Objetivos del estudio

- Comprender fundamentos de redes y protocolos
- Aplicar técnicas de reconocimiento y enumeración
- Identificar y explotar vulnerabilidades web básicas
- Realizar escaladas de privilegios en entornos controlados
- Consolidar metodología de pentesting

---

## 🛠 Herramientas utilizadas

- Nmap
- Burp Suite
- Metasploit
- Netcat
- Gobuster / ffuf
- Wireshark
- Msfvenom
- Bash & Python
